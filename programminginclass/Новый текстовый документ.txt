Функции (пользовательские)
(лирическое отступление: 2 ** 3 - 2 в кубе)
Пользовательские функции нужны, чтобы:
 1. удобно представлять в программе места, где используется один и тот же код.
 2. удобно фиксить (один раз в программе, а не n раз).

Синтаксис:
def - объявление (опеределение) функции
def func1(a, b, n) :
    func1() :

def func1(a, b) :
	c = a + b
	d = c ** 3
	print(d)
	return d - завершает работу функции (да, мы не ставим тут скобочки)

a = func1(x,y) - вызов функции (да, мы можем делать две разные а в теле программы и теле функции и нельзя вызвать переменную из функции напрямую в теле программы)
func2(x) - вызов функции, которая ничего не возвращает

Сколько аргументов при вызове, столько должно быть при объявлении (ну это пока на нашем поверхностном уровне)

NoneType - значение, присваиваемое переменной, если в функции по ошибке нет return
